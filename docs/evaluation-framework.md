# QTEC Knowledge Brain — Evaluation Framework

## Test Question Set

### หมวดนโยบายการลา (Employee Handbook)
1. นโยบายการลาพักร้อนเป็นอย่างไร
2. ลาป่วยต้องทำอย่างไร มีเงื่อนไขอะไรบ้าง
3. พนักงานใหม่ต้องทดลองงานกี่วัน
4. สิทธิ์วันลาพักร้อนเพิ่มตามอายุงานอย่างไร

### หมวดค่าล่วงเวลา (Employee Handbook)
5. ขั้นตอนเบิกค่า OT ต้องทำอย่างไร
6. ถ้าส่งเอกสาร OT ล่าช้าจะเกิดอะไรขึ้น
7. OT ต้องได้รับอนุมัติจากใคร

### หมวดสวัสดิการ (Employee Handbook + สถานพยาบาล)
8. สวัสดิการพนักงานมีอะไรบ้าง
9. ค่ารักษาพยาบาลเบิกได้เท่าไร
10. ตรวจสุขภาพประจำปีทำที่ไหนได้

### หมวดวันหยุดประจำปี (ประกาศ 002/2568)
11. วันหยุดประจำปี 2569 มีกี่วัน
12. สงกรานต์ปี 2569 หยุดวันไหนถึงวันไหน
13. วันหยุดเดือนเมษายนมีวันไหนบ้าง
14. มีวันหยุดชดเชยวันไหนบ้าง

### หมวดสถานพยาบาล (รายชื่อสถานพยาบาล 2569)
15. โรงพยาบาลคู่สัญญามีที่ไหนบ้าง
16. คลินิกทันตกรรมที่ใช้สิทธิ์ได้มีที่ไหน
17. สถานพยาบาลประกันสังคมของบริษัทคือที่ไหน

### หมวดจรรยาบรรณ (Employee Handbook)
18. จรรยาบรรณพนักงานมีอะไรบ้าง
19. ข้อห้ามที่สำคัญของพนักงานมีอะไร

### Edge Cases / Out-of-Scope
20. ราคาหุ้นของบริษัทเป็นเท่าไร (ควรปฏิเสธ)
21. เมนูอาหารกลางวันวันนี้มีอะไร (ควรปฏิเสธ)
22. Who is the CEO? (ภาษาอังกฤษ — ควรตอบไทยจากเอกสาร)

## Evaluation Metrics

| Metric | เป้าหมาย PoC | วิธีวัด |
|--------|:------------:|---------|
| Answer Accuracy | ≥ 80% | ตรงกับเอกสารจริง |
| Grounding Coverage | ≥ 90% | มี source citation |
| Latency (mock) | < 2 วินาที | `latencyMs` ใน response |
| Latency (live) | < 5 วินาที | `latencyMs` ใน response |
| Unanswered Rate | ≤ 10% | ตอบ "ไม่พบ" เมื่อมีข้อมูลจริงอยู่ |
| Rejection Rate | 100% สำหรับ out-of-scope | ปฏิเสธคำถามนอกขอบเขต |

## UAT Checklist

- [ ] ทดสอบกับ HR — นโยบายการลา, สวัสดิการ, OT
- [ ] ทดสอบกับ QA — คู่มือความปลอดภัย
- [ ] ทดสอบกับ IT — ทดสอบ edge cases
- [ ] ตรวจ source citations ตรงกับเอกสารจริง
- [ ] ตรวจ latency อยู่ในเกณฑ์
- [ ] ตรวจ out-of-scope rejection ทำงานถูกต้อง
